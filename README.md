# Cards

## Elixir Install
% brew install elixir

## 対話モード
- IExという対話シェルがElixirにはある
- 対話モード開始コマンド→ iex -S mix
- 対話モード終了コマンド→ control + c

## 基礎データ型
- 整数(Integers) iex> 217
- 浮動小数(Floats) iex> 3.12
- 真理値(Booleans) iex> true / false
- アトム(Atom)
  iex> :foo
  :foo
  iex> :foo == :bar
  false

  iex> is_atom(true)
  true
  iex> :true == true
  true
- 文字列(Strings) iex> "Hello"
